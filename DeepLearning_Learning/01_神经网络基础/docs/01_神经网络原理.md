# 第一章：神经网络基础

## 1.1 什么是神经网络？

神经网络是一种模拟人脑神经元工作方式的计算模型，由大量的节点（神经元）相互连接组成。

### 1.1.1 感知机（Perceptron）

感知机是最简单的神经网络，只有一个神经元：

```
输入 x1 → w1 ↘
输入 x2 → w2 → ∑(加权求和) → 激活函数 → 输出
输入 x3 → w3 ↗
```

**数学表达**：
```
y = f(w1*x1 + w2*x2 + w3*x3 + b)
```
其中：
- x1, x2, x3 是输入
- w1, w2, w3 是权重
- b 是偏置（bias）
- f 是激活函数

### 1.1.2 多层感知机（MLP）

多层感知机由多个层组成：
- **输入层**：接收原始数据
- **隐藏层**：提取特征（可以有多个）
- **输出层**：输出预测结果

```
输入层      隐藏层       输出层
  ○ ────────→ ○ ────────→ ○
  ○ ────────→ ○ ────────→ ○
  ○ ────────→ ○
```

## 1.2 前向传播（Forward Propagation）

前向传播是数据从输入层经过隐藏层到输出层的过程。

### 计算步骤：

1. **输入层 → 隐藏层**：
   ```
   z1 = W1 · x + b1
   a1 = activation(z1)
   ```

2. **隐藏层 → 输出层**：
   ```
   z2 = W2 · a1 + b2
   output = activation(z2)
   ```

### 示例代码：

```python
import numpy as np

def forward_propagation(X, W1, b1, W2, b2):
    """
    前向传播
    X: 输入数据 (n_features, m_samples)
    W1: 第一层权重 (n_hidden, n_features)
    b1: 第一层偏置 (n_hidden, 1)
    W2: 第二层权重 (n_output, n_hidden)
    b2: 第二层偏置 (n_output, 1)
    """
    # 隐藏层
    Z1 = np.dot(W1, X) + b1
    A1 = sigmoid(Z1)
    
    # 输出层
    Z2 = np.dot(W2, A1) + b2
    A2 = sigmoid(Z2)
    
    cache = {"Z1": Z1, "A1": A1, "Z2": Z2, "A2": A2}
    
    return A2, cache

def sigmoid(z):
    """Sigmoid激活函数"""
    return 1 / (1 + np.exp(-z))
```

## 1.3 激活函数

激活函数为神经网络引入非线性，使其能够学习复杂的模式。

### 1.3.1 常用激活函数

#### Sigmoid
```
f(x) = 1 / (1 + e^(-x))
```
- 输出范围：(0, 1)
- 优点：平滑、可微
- 缺点：梯度消失问题

#### ReLU（Rectified Linear Unit）
```
f(x) = max(0, x)
```
- 输出范围：[0, +∞)
- 优点：计算快、缓解梯度消失
- 缺点：Dead ReLU问题

#### Tanh
```
f(x) = (e^x - e^(-x)) / (e^x + e^(-x))
```
- 输出范围：(-1, 1)
- 优点：零中心化
- 缺点：梯度消失

#### Softmax（用于多分类输出层）
```
f(xi) = e^xi / Σ(e^xj)
```
- 输出范围：(0, 1)，所有输出和为1
- 用途：多分类问题

### 1.3.2 激活函数选择建议

- **隐藏层**：ReLU（最常用）
- **输出层**：
  - 二分类：Sigmoid
  - 多分类：Softmax
  - 回归：线性激活（不使用激活函数）

## 1.4 损失函数

损失函数衡量模型预测与真实值之间的差距。

### 1.4.1 常用损失函数

#### 均方误差（MSE）- 回归问题
```
L = (1/m) * Σ(yi - ŷi)^2
```

#### 交叉熵损失（Cross-Entropy）- 分类问题
```
L = -(1/m) * Σ[yi*log(ŷi) + (1-yi)*log(1-ŷi)]
```

### 1.4.2 代码实现

```python
def compute_loss(Y, Y_pred, loss_type="cross_entropy"):
    """
    计算损失
    Y: 真实标签
    Y_pred: 预测值
    loss_type: 损失函数类型
    """
    m = Y.shape[1]
    
    if loss_type == "cross_entropy":
        # 防止log(0)
        epsilon = 1e-15
        Y_pred = np.clip(Y_pred, epsilon, 1 - epsilon)
        loss = -np.sum(Y * np.log(Y_pred) + (1 - Y) * np.log(1 - Y_pred)) / m
    elif loss_type == "mse":
        loss = np.sum((Y - Y_pred) ** 2) / m
    
    return loss
```

## 1.5 反向传播（Backpropagation）

反向传播是神经网络训练的核心算法，通过链式法则计算梯度。

### 1.5.1 核心思想

从输出层向输入层反向传播误差，计算每个参数的梯度，然后更新参数。

### 1.5.2 链式法则

```
∂L/∂W = ∂L/∂a * ∂a/∂z * ∂z/∂W
```

### 1.5.3 反向传播步骤

```python
def backward_propagation(X, Y, cache, parameters):
    """
    反向传播
    """
    m = X.shape[1]
    
    W1 = parameters["W1"]
    W2 = parameters["W2"]
    
    A1 = cache["A1"]
    A2 = cache["A2"]
    
    # 输出层梯度
    dZ2 = A2 - Y
    dW2 = np.dot(dZ2, A1.T) / m
    db2 = np.sum(dZ2, axis=1, keepdims=True) / m
    
    # 隐藏层梯度
    dZ1 = np.dot(W2.T, dZ2) * sigmoid_derivative(A1)
    dW1 = np.dot(dZ1, X.T) / m
    db1 = np.sum(dZ1, axis=1, keepdims=True) / m
    
    gradients = {
        "dW1": dW1, "db1": db1,
        "dW2": dW2, "db2": db2
    }
    
    return gradients

def sigmoid_derivative(A):
    """Sigmoid导数"""
    return A * (1 - A)
```

## 1.6 参数更新（优化算法）

### 1.6.1 梯度下降

```python
def update_parameters(parameters, gradients, learning_rate):
    """
    使用梯度下降更新参数
    """
    W1 = parameters["W1"] - learning_rate * gradients["dW1"]
    b1 = parameters["b1"] - learning_rate * gradients["db1"]
    W2 = parameters["W2"] - learning_rate * gradients["dW2"]
    b2 = parameters["b2"] - learning_rate * gradients["db2"]
    
    return {"W1": W1, "b1": b1, "W2": W2, "b2": b2}
```

### 1.6.2 优化算法对比

| 算法 | 特点 | 适用场景 |
|------|------|---------|
| SGD | 简单、稳定 | 小数据集 |
| Momentum | 加速收敛 | 深层网络 |
| Adam | 自适应学习率 | 大多数场景 |
| RMSprop | 自适应学习率 | RNN网络 |

## 1.8 重点概念总结

### ✅ 必须理解的概念
1. **前向传播**：数据从输入到输出的过程
2. **反向传播**：误差从输出到输入的传播
3. **激活函数**：引入非线性
4. **损失函数**：衡量预测误差
5. **梯度下降**：优化参数的方法

### ✅ 常见问题
1. **梯度消失**：使用ReLU激活函数
2. **过拟合**：使用Dropout、正则化
3. **训练慢**：使用Adam优化器、批归一化
