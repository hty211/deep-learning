# 深度学习基础问题

## 1. 神经网络基础

### 1.1 激活函数

**Q: 常用的激活函数有哪些？各有什么优缺点？**

```
1. Sigmoid
   - 公式: σ(x) = 1/(1+e^(-x))
   - 优点: 输出在(0,1)，适合概率输出
   - 缺点: 梯度消失、输出非零中心、计算exp较慢

2. Tanh
   - 公式: tanh(x) = (e^x - e^(-x))/(e^x + e^(-x))
   - 优点: 输出在(-1,1)，零中心
   - 缺点: 梯度消失

3. ReLU
   - 公式: f(x) = max(0, x)
   - 优点: 计算快、缓解梯度消失、稀疏激活
   - 缺点: Dead ReLU问题

4. Leaky ReLU
   - 公式: f(x) = max(αx, x), α通常为0.01
   - 优点: 解决Dead ReLU
   - 缺点: 需要调参

5. GELU
   - 公式: x * Φ(x)，Φ是标准正态分布CDF
   - 优点: 在Transformer中效果好
   - 缺点: 计算复杂

6. Swish
   - 公式: x * sigmoid(x)
   - 优点: 平滑、无上界有下界
   - 缺点: 计算较慢
```

### 1.2 损失函数

**Q: 回归和分类常用的损失函数有哪些？**

```
回归:
1. MSE (均方误差): L = (1/n)Σ(y-ŷ)²
   - 对异常值敏感
   
2. MAE (平均绝对误差): L = (1/n)Σ|y-ŷ|
   - 对异常值鲁棒
   
3. Huber Loss: MSE和MAE的结合
   - 小误差用MSE，大误差用MAE

分类:
1. 交叉熵损失: L = -Σy*log(ŷ)
   - 常用于多分类
   
2. 二元交叉熵: L = -[y*log(ŷ) + (1-y)*log(1-ŷ)]
   - 用于二分类
   
3. Focal Loss: L = -α(1-ŷ)^γ * y*log(ŷ)
   - 解决类别不平衡
```

### 1.3 反向传播

**Q: 解释反向传播算法的原理**

```
反向传播是计算梯度的算法，基于链式法则：

1. 前向传播: 计算每层的输出
2. 计算损失: L = loss(output, target)
3. 反向传播: 从输出层向输入层计算梯度
   - ∂L/∂w = ∂L/∂y * ∂y/∂w
   - 逐层应用链式法则
4. 参数更新: w = w - η * ∂L/∂w

关键点:
- 计算图自动微分
- 梯度的链式传播
- 内存换时间策略
```

## 2. 优化算法

### 2.1 梯度下降变体

**Q: SGD、Momentum、Adam的区别？**

```
1. SGD (随机梯度下降)
   - w = w - η * g
   - 简单但震荡
   
2. Momentum
   - v = βv + g
   - w = w - η * v
   - 加速收敛，减少震荡
   
3. NAG (Nesterov)
   - 先预测再计算梯度
   - 更智能的动量
   
4. AdaGrad
   - 自适应学习率
   - 累积梯度平方
   
5. RMSprop
   - 解决AdaGrad学习率衰减问题
   - 使用滑动平均
   
6. Adam
   - 结合Momentum和RMSprop
   - m = β₁m + (1-β₁)g
   - v = β₂v + (1-β₂)g²
   - w = w - η * m/(√v + ε)
   - 最常用的优化器
```

### 2.2 学习率调度

**Q: 常用的学习率调度策略？**

```
1. Step Decay: 每N个epoch降低学习率
   lr = lr0 * decay^step

2. Exponential Decay: 指数衰减
   lr = lr0 * e^(-kt)

3. Cosine Annealing: 余弦退火
   lr = lr_min + 0.5(lr_max - lr_min)(1 + cos(πt/T))

4. Warmup: 先预热再正常调度
   - 防止初期梯度不稳定
   - 常用于Transformer训练

5. One Cycle Policy: 一个周期策略
   - 先增后减
   - 超收敛效果
```

### 3.2 Batch Normalization

**Q: 解释Batch Normalization的原理和作用**

```
原理:
1. 计算mini-batch的均值和方差
   μ_B = (1/m)Σx_i
   σ²_B = (1/m)Σ(x_i - μ_B)²

2. 标准化
   x̂_i = (x_i - μ_B) / √(σ²_B + ε)

3. 缩放和平移
   y_i = γ * x̂_i + β

作用:
- 加速训练收敛
- 允许使用更大学习率
- 减少对初始化的依赖
- 一定的正则化效果

训练和推理的区别:
- 训练: 使用当前batch统计量
- 推理: 使用训练时的滑动平均统计量
```

### 3.3 Dropout

**Q: Dropout的原理？训练和推理的区别？**

```
原理:
- 训练时以概率p随机"丢弃"神经元
- 相当于训练多个子网络的集成
- 推理时使用全部神经元，输出乘以(1-p)

为什么有效:
- 防止神经元共适应
- 模型集成效果
- 增加鲁棒性

变体:
- DropConnect: 随机丢弃连接
- Spatial Dropout: 整个特征图丢弃
- DropBlock: 块状丢弃
```

## 4. 初始化

**Q: 为什么权重初始化重要？常用的初始化方法？**

```
为什么重要:
- 影响梯度传播
- 影响收敛速度
- 影响最终性能

常用方法:

1. Xavier初始化
   - 适用于Sigmoid/Tanh
   - Var(w) = 2/(n_in + n_out)
   
2. He初始化 (Kaiming)
   - 适用于ReLU
   - Var(w) = 2/n_in
   
3. 预训练初始化
   - 使用预训练权重
   - 迁移学习常用
```

## 5. 卷积神经网络

### 5.1 卷积基础

**Q: 卷积层的关键参数？**

```
1. 卷积核大小 (Kernel Size)
   - 常用: 3x3, 5x5, 7x7
   - 小卷积堆叠效果好

2. 步长 (Stride)
   - 控制输出大小
   - 步长>1时下采样

3. 填充 (Padding)
   - 保持特征图大小
   - 'same' vs 'valid'

4. 输出大小计算:
   H_out = (H_in + 2P - K) / S + 1
   
5. 感受野:
   - 堆叠小卷积增大感受野
   - 两个3x3等价于一个5x5
```

### 5.2 池化层

**Q: 池化层的作用？**

```
作用:
- 降维，减少计算量
- 增大感受野
- 提供平移不变性

类型:
1. Max Pooling: 取最大值
   - 保留主要特征
   
2. Average Pooling: 取平均值
   - 更平滑
   
3. Global Average Pooling: 全局平均
   - 替代全连接层
   - 减少参数
```

## 6. 循环神经网络

### 6.1 RNN问题

**Q: RNN的梯度消失问题如何解决？**

```
问题原因:
- 长序列中梯度连乘
- 激活函数导数<1时梯度消失
- 激活函数导数>1时梯度爆炸

解决方案:

1. LSTM/GRU
   - 门控机制控制信息流
   - 遗忘门、输入门、输出门
   
2. 梯度裁剪
   - 限制梯度范数
   - 防止梯度爆炸
   
3. 更好的初始化
   - 单位矩阵初始化
   
4. 使用ReLU + 良好初始化
```

### 6.2 LSTM

**Q: 解释LSTM的门控机制**

```
LSTM三个门:

1. 遗忘门 (Forget Gate)
   f_t = σ(W_f * [h_{t-1}, x_t] + b_f)
   - 决定丢弃哪些信息

2. 输入门 (Input Gate)
   i_t = σ(W_i * [h_{t-1}, x_t] + b_i)
   C̃_t = tanh(W_C * [h_{t-1}, x_t] + b_C)
   - 决定存储哪些新信息

3. 输出门 (Output Gate)
   o_t = σ(W_o * [h_{t-1}, x_t] + b_o)
   h_t = o_t * tanh(C_t)
   - 决定输出哪些信息

细胞状态更新:
C_t = f_t * C_{t-1} + i_t * C̃_t
```

## 7. 注意力机制

**Q: 注意力机制的核心思想？**

```
核心公式:
Attention(Q, K, V) = softmax(QK^T / √d_k) V

组件:
- Query (Q): 查询向量
- Key (K): 键向量
- Value (V): 值向量

计算步骤:
1. 计算相关性分数: QK^T
2. 缩放: 除以√d_k
3. 归一化: softmax
4. 加权求和: 乘以V

多头注意力:
- 多个子空间并行计算
- 捕获不同类型的依赖关系
```
