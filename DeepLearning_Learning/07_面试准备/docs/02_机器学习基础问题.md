# 机器学习基础问题

## 1. 监督学习

### 1.1 分类与回归

**Q: 分类和回归的区别？**

```
分类:
- 输出离散标签
- 评估指标: 准确率、精确率、召回率、F1
- 损失函数: 交叉熵

回归:
- 输出连续值
- 评估指标: MSE、MAE、R²
- 损失函数: 均方误差
```

### 1.2 评估指标

**Q: 解释精确率、召回率、F1**

```
混淆矩阵:
                预测正类    预测负类
实际正类          TP          FN
实际负类          FP          TN

精确率 (Precision) = TP / (TP + FP)
- 预测为正的样本中，实际为正的比例
- 关注预测的准确性

召回率 (Recall) = TP / (TP + FN)
- 实际为正的样本中，预测为正的比例
- 关注覆盖的完整性

F1 = 2 * (Precision * Recall) / (Precision + Recall)
- 精确率和召回率的调和平均
- 平衡两者的指标

ROC曲线:
- TPR (True Positive Rate) = Recall
- FPR (False Positive Rate) = FP / (FP + TN)
- AUC: ROC曲线下面积

PR曲线:
- 精确率-召回率曲线
- 类别不平衡时更有参考价值
```

## 2. 偏差与方差

**Q: 解释偏差和方差，如何权衡？**

```
偏差 (Bias):
- 模型预测与真实值的差距
- 高偏差 = 欠拟合
- 解决: 增加模型复杂度、添加特征

方差 (Variance):
- 模型对数据变化的敏感度
- 高方差 = 过拟合
- 解决: 正则化、简化模型、增加数据

偏差-方差权衡:
总误差 = 偏差² + 方差 + 噪声

高偏差情况:
- 训练误差高，验证误差高
- 欠拟合

高方差情况:
- 训练误差低，验证误差高
- 过拟合

解决方案:
| 问题 | 解决方案 |
|------|---------|
| 高偏差 | 增加特征、更复杂模型、减少正则化 |
| 高方差 | 更多数据、正则化、简化模型、特征选择 |
```

## 3. 正则化

### 3.1 L1 vs L2

**Q: L1和L2正则化的区别？**

```
L1正则化 (Lasso):
- 损失函数加: λΣ|w|
- 产生稀疏解
- 可用于特征选择
- 几何解释: 菱形约束

L2正则化 (Ridge):
- 损失函数加: λΣw²
- 权重衰减
- 不会产生精确零
- 几何解释: 圆形约束

为什么L1产生稀疏解:
- L1的等高线与约束区域交点在轴上
- 容易使某些权重精确为零

Elastic Net:
- 结合L1和L2: λ₁Σ|w| + λ₂Σw²
- 兼具两者优点
```

## 4. 决策树与集成学习

### 4.1 决策树

**Q: 决策树的分裂标准？**

```
1. 信息增益 (ID3)
   - 基于信息熵
   - IG = H(D) - H(D|A)
   - 偏向取值多的特征

2. 信息增益率 (C4.5)
   - 解决ID3偏向问题
   - IGR = IG / H(A)

3. 基尼指数 (CART)
   - Gini = 1 - Σp_i²
   - 分类树常用
   - 计算更快

4. 均方误差 (CART回归)
   - 回归树常用
   - 最小化MSE
```

### 4.2 集成学习

**Q: Bagging和Boosting的区别？**

```
Bagging (如随机森林):
- 并行训练多个模型
- 有放回采样
- 减少方差
- 对噪声鲁棒

Boosting (如XGBoost):
- 串行训练，后者纠正前者
- 加权组合
- 减少偏差
- 对噪声敏感

随机森林:
- 多棵决策树投票
- 特征随机选择
- 并行训练

XGBoost:
- 梯度提升框架
- 二阶泰勒展开
- 正则化
- 并行化特征选择
```

## 5. 支持向量机

**Q: SVM的原理？核函数的作用？**

```
SVM原理:
- 寻找最大间隔超平面
- 优化目标: max margin = 2/||w||
- 约束: y_i(w·x_i + b) ≥ 1

软间隔:
- 允许部分样本分类错误
- 引入松弛变量ξ
- 目标: min ||w||² + CΣξ_i

核函数:
- 将低维映射到高维
- 使线性不可分变得可分
- 常用核:
  - 线性核: K(x,z) = x·z
  - 多项式核: K(x,z) = (x·z + c)^d
  - RBF核: K(x,z) = exp(-γ||x-z||²)

核技巧:
- 直接计算高维内积
- 不显式计算映射
```

## 6. 聚类

**Q: K-Means算法原理？如何选择K？**

```
K-Means步骤:
1. 随机初始化K个中心点
2. 分配每个点到最近的中心
3. 重新计算中心位置
4. 重复2-3直到收敛

选择K值:
1. 肘部法则
   - 绘制SSE-K曲线
   - 选择拐点

2. 轮廓系数
   - 衡量聚类质量
   - 范围[-1, 1]，越大越好

3. Gap Statistic
   - 比较实际与随机分布

K-Means++初始化:
- 第一个中心随机选择
- 后续中心选择距离远的点
- 避免局部最优
```

## 7. 降维

**Q: PCA的原理？**

```
PCA步骤:
1. 数据中心化
2. 计算协方差矩阵
3. 计算特征值和特征向量
4. 选择前k个最大特征值对应的特征向量
5. 投影到新空间

原理:
- 寻找方差最大的方向
- 数据在新方向上的投影保留最多信息

数学推导:
- 协方差矩阵C = X^T X / n
- 特征分解: C = VΛV^T
- 选择最大特征值对应的方向

PCA vs LDA:
- PCA: 无监督，最大化方差
- LDA: 有监督，最大化类间距离
```

## 8. 特征工程

**Q: 常用的特征工程方法？**

```
1. 数值特征:
   - 标准化: (x - μ) / σ
   - 归一化: (x - min) / (max - min)
   - 分箱: 连续转离散
   - 多项式特征

2. 类别特征:
   - One-Hot编码
   - Label编码
   - 目标编码
   - 嵌入编码

3. 文本特征:
   - TF-IDF
   - 词嵌入
   - N-gram

4. 特征选择:
   - 过滤法: 相关性、卡方检验
   - 包装法: 前向/后向选择
   - 嵌入法: L1正则、树模型重要性

5. 特征构造:
   - 交互特征
   - 统计特征
   - 时间特征
```

## 9. 模型评估

**Q: 交叉验证的作用？常用的交叉验证方法？**

```
作用:
- 更准确评估模型性能
- 充分利用数据
- 减少过拟合风险

常用方法:

1. K折交叉验证
   - 数据分成K份
   - 轮流作为验证集
   - 取平均结果

2. 留一交叉验证 (LOOCV)
   - K = N
   - 每次留一个样本验证
   - 计算量大但最准确

3. 分层K折
   - 保持每折的类别比例
   - 类别不平衡时使用

4. 时间序列交叉验证
   - 按时间划分
   - 训练集在验证集之前
   - 时间序列数据使用
```

## 10. 数据不平衡

**Q: 如何处理类别不平衡问题？**

```
1. 重采样:
   - 过采样少数类 (SMOTE)
   - 欠采样多数类
   - 混合采样

2. 代价敏感学习:
   - 调整损失函数权重
   - 少数类错误惩罚更大

3. 阈值调整:
   - 调整分类阈值
   - 优化F1或其他指标

4. 集成方法:
   - BalancedRandomForest
   - EasyEnsemble

5. 数据增强:
   - 对少数类进行增强
   - 生成合成样本

6. 评估指标:
   - 使用F1、AUC而非准确率
   - 关注少数类性能
```
