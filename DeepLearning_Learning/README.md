## 🎯 学习方式

每个模块包含以下内容：

```
模块目录/
├── 学习指南.md      # 学习路径和目标
├── docs/            # 理论文档
├── examples/        # 代码示例
└── exercises/       # 练习题
```

### 学习步骤

1. **阅读学习指南**：了解模块目标和学习路径
2. **理论学习**：阅读 `docs/` 目录下的教程文档
3. **代码实践**：运行 `examples/` 目录下的代码示例
4. **练习巩固**：完成 `exercises/` 目录下的练习题
